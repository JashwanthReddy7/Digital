# ⚖️ LegalPro+

A modern and simple **Digital Legal Marketplace** web application built using **HTML, CSS, and JavaScript**.
This project allows users to post legal issues, explore top lawyers, and simulate a payment process.

---

## 🚀 Features

* 🎯 Attractive Hero Section (Landing Page)
* 📝 Post Legal Issues
* 👨‍⚖️ Browse Top Lawyers
* 💳 Payment Popup (Demo)
* ✅ Payment Success Alert
* 📱 Responsive UI Design

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling & Layout
* **JavaScript (Vanilla JS)** – Functionality

## ▶️ How to Run

### Option 1: Using VS Code Live Server

1. Open project in VS Code
2. Install **Live Server Extension**
3. Click **"Go Live"**
4. Open in browser:
   `http://127.0.0.1:5500/index.html`

### Option 2: Open Directly

* Double-click `index.html`
* Opens in your default browser

---

## 💡 How It Works

* Click **"Get Started"**
* Scroll and **Post Legal Issue**
* Click **Contact**
* Payment popup appears
* Enter dummy details → Click **Pay**
* Shows **Payment Successful**

---

## ⚠️ Note

* This is a **frontend-only demo project**
* Payment system is **not real (simulation only)**
* No backend/database is connected
